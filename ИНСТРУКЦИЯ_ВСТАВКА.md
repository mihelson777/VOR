# Куда вставить API ключ

## Шаг 1 — Создай файл `.env`

В корне проекта (рядом с `config.py`) создай файл **`.env`**.

## Шаг 2 — Вставь это (и замени ключ)

Скопируй в `.env`:

```
GROQ_API_KEY=твой_ключ_сюда
WEB_PASSWORD=vor
```

**Где взять ключ:** https://console.groq.com/keys (бесплатно, без карты)

## Шаг 3 — Перезапусти

```powershell
python web_server.py
```

или

```powershell
python telegram_bot.py
```

---

**Альтернатива:** открой `PASTE_СЮДА.env`, вставь ключ вместо `ВСТАВЬ_КЛЮЧ_СЮДА`, переименуй файл в `.env`.
