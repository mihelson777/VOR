# Бесплатные API для агента — топ варианты

Все перечисленные варианты **без привязки карты** (кроме явно платных).

---

## 1. Groq (рекомендую)

**Плюсы:** очень быстрый ответ, большой бесплатный лимит, OpenAI-совместимый API.

| Что | Значение |
|-----|----------|
| Лимит | ~14 400 запросов/день, ~30 запросов/минуту |
| Модели | Llama 3.3 70B, Llama 3.1 70B/8B, Mixtral 8x7B, Gemma2 9B |
| Контекст | до 128K токенов |
| Карта | не нужна |

**Как получить ключ:**
1. Зайди на **[console.groq.com](https://console.groq.com)**
2. Войди через Google/GitHub
3. **API Keys** → **Create API Key** → скопируй ключ

**В проекте:**
```powershell
$env:GROQ_API_KEY = "gsk_..."
$env:OUROBOROS_MODEL = "llama-3.3-70b-versatile"
python main.py
```

---

## 2. OpenRouter (бесплатный роутер)

**Плюсы:** один ключ — доступ ко многим бесплатным моделям через роутер `openrouter/free`.

| Что | Значение |
|-----|----------|
| Модель в коде | `openrouter/free` (роутер сам выбирает модель) |
| Контекст | до 200K токенов |
| Карта | не нужна для бесплатных моделей |

**Как получить ключ:**
1. Зайди на **[openrouter.ai](https://openrouter.ai)**
2. Войди (Google и т.д.)
3. **Keys** → **Create Key** → скопируй

**В проекте:**
```powershell
$env:OPENROUTER_API_KEY = "sk-or-..."
$env:OUROBOROS_MODEL = "openrouter/free"
python main.py
```

---

## 3. Google Gemini

**Плюсы:** официальный Google API, щедрый бесплатный tier.

| Что | Значение |
|-----|----------|
| Gemini 2.5 Flash | до 500 запросов/день |
| Gemini 2.5 Pro | до 50 запросов/день |
| Карта | не нужна |

**Как получить ключ:**
1. Зайди на **[aistudio.google.com/apikey](https://aistudio.google.com/apikey)** (или makersuite.google.com/app/apikey)
2. Войди в Google-аккаунт
3. **Create API Key** → скопируй (начинается с `AIza...`)

**В проекте:** используй через OpenRouter как `google/gemini-2.0-flash-001` (или другой free-модель) с ключом OpenRouter, либо отдельная поддержка Gemini в коде (см. ниже).

---

## 4. Другие варианты

- **Anthropic (Claude):** [console.anthropic.com](https://console.anthropic.com) — при регистрации дают **$5** бесплатно, дальше по карте.
- **OpenAI:** бесплатного tier для API почти нет, только платно.
- **ApiFreeLLM / LLMAPI.AI:** сторонние агрегаторы с лимитами и очередями — можно гуглить по названию, если нужен запасной вариант.

---

## Что поставить в первую очередь

1. **Groq** — один ключ, быстрый ответ, большие лимиты.  
2. **OpenRouter** — если хочешь пробовать разные бесплатные модели через один ключ.

В `config.py` и `ouroboros/llm.py` добавлена поддержка **GROQ_API_KEY** и модели по умолчанию для бесплатного запуска.
